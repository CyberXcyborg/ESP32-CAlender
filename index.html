Installation Instructions

What You'll Need:

  • ESP32 development board
  • Micro USB cable
  • Arduino IDE installed
  • ESP32 board definition in Arduino IDE
  • Required libraries:
    • WiFi
    • WebServer
    • SPIFFS
    • ArduinoJson

Quick Setup:

  1. Clone the GitHub repository
  2. Open the project in Arduino IDE
  3. Update WiFi credentials in the code
  4. Select the correct board and port in Arduino IDE
  5. Upload the code to your ESP32
  6. Open Serial Monitor at 115200 baud to see the IP address
  7. Access the calendar interface at http://[IP_ADDRESS]
  8. Start adding events to your calendar!

// Event Management Functions
function addEvent() {
  const text = document.getElementById("eventText").value.trim();
  const time = document.getElementById("eventTime").value;
  const category = document.getElementById("eventCategory").value;

  if (!text) {
    alert("Please enter event description.");
    return;
  }

  if (!events[selectedDate]) {
    events[selectedDate] = [];
  }

  events[selectedDate].push({ text, time, category });
  
  // Save to server
  saveEvents();
  
  document.getElementById("eventText").value = '';
  
  updateEventsList();
  renderCalendar();
}

// Save events to ESP32 SPIFFS
function saveEvents() {
  const saveData = async (retries = 3) => {
    try {
      const response = await fetch('/events', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json'
        },
        body: JSON.stringify(events)
      });
      
      if (!response.ok) {
        console.error('Failed to save events');
        if (retries > 0) {
          setTimeout(() => saveData(retries - 1), 1000);
        }
      }
    } catch (error) {
      console.error('Error saving events:', error);
      if (retries > 0) {
        setTimeout(() => saveData(retries - 1), 1000);
      }
    }
  };
  
  saveData();
}

Key Implementation Features

SPIFFS Storage

Events are stored in JSON format on the ESP32's SPIFFS file system, ensuring data persistence across power cycles.

WebServer Integration

The ESP32 serves the calendar interface via a built-in web server, making it accessible from any device on your network.

Responsive UI

The interface is built with modern HTML, CSS, and JavaScript, providing a smooth user experience across all devices.

Efficient Resource Usage

The calendar is optimized for ESP32's limited resources, with careful memory management and efficient data handling.

Server Configuration

The ESP32 web server runs on port 80 by default. You can access the calendar interface by typing the ESP32's IP address into your browser.
